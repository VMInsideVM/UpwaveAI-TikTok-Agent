# 响应式设计说明

## 已实现的窗口适配功能

### 桌面端（> 1024px）
- 完整侧边栏（280px宽度）
- 三栏布局：侧边栏 + 聊天区 + 顶部栏
- 消息最大宽度 70%
- 完整的用户信息显示

### 平板端（768px - 1024px）
- 侧边栏缩小至 240px
- 内边距适当缩小
- 消息最大宽度保持 70%
- 所有功能正常显示

### 移动端（< 768px）
- **侧边栏自动隐藏**：默认隐藏在屏幕左侧
- **汉堡菜单按钮**：点击顶部左侧的 ☰ 按钮展开/收起侧边栏
- **半透明遮罩层**：侧边栏打开时，主内容区域显示遮罩，点击可关闭侧边栏
- **优化间距**：减小 padding 和字体大小
- **消息宽度**：增加到 85%
- **按钮自适应**：缩小按钮和输入框尺寸

### 小屏手机（< 480px）
- **隐藏标题**：顶部栏标题隐藏，节省空间
- **隐藏用户名**：仅显示用户图标
- **精简配额显示**：更小的字体和内边距
- **消息宽度**：增加到 90%
- **模态框全屏**：报告库等模态框占据 95% 宽度
- **单列显示**：报告网格改为单列布局

## 新增功能

### 1. 侧边栏切换
```javascript
function toggleSidebar() {
    const sidebar = document.getElementById('sidebar');
    const overlay = document.getElementById('sidebarOverlay');
    sidebar.classList.toggle('show');
    overlay.classList.toggle('show');
}
```

### 2. 遮罩层
- 移动端打开侧边栏时显示半透明黑色遮罩
- 点击遮罩可关闭侧边栏
- 桌面端不显示

### 3. 滚动条美化
- 宽度 6px
- 圆角设计
- 悬停时变色
- 适配深色主题

### 4. 文字换行优化
```css
.message-bubble {
    overflow-wrap: break-word;
    word-break: break-word;
}
```
防止长文本溢出

## 响应式断点

| 屏幕宽度 | 布局调整 |
|---------|---------|
| > 1024px | 桌面端完整布局 |
| 768px - 1024px | 平板端缩小布局 |
| 480px - 768px | 移动端隐藏侧边栏 |
| < 480px | 小屏手机精简显示 |

## 使用方法

### 移动端打开侧边栏
1. 点击左上角 **☰** 按钮
2. 侧边栏从左侧滑出
3. 主内容区域显示遮罩
4. 点击遮罩或再次点击 ☰ 可关闭

### 查看报告库
- 桌面端：报告以网格形式显示（3列）
- 移动端：报告以单列形式显示
- 模态框自动适配屏幕大小
