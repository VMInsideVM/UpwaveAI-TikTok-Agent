## 常见问题排查

### 问题 1：上传图片后无响应

**可能原因**：
- Playwright API 服务未启动
- Chrome CDP 未连接
- 网络连接问题

**解决方法**：
```bash
# 检查 Playwright API 服务
curl http://127.0.0.1:8000/health

# 重启 Playwright API
# 终端 1
python start_api.py
```

### 问题 2：Agent 返回错误"无法处理图片"

**可能原因**：
- IMAGE_MODEL 配置错误
- API key 无效
- 视觉模型不支持

**解决方法**：
检查 `.env` 文件：
```env
IMAGE_MODEL="Qwen/Qwen3-VL-235B-A22B-Thinking"
OPENAI_API_KEY="sk-xxx"
OPENAI_BASE_URL="https://api.siliconflow.cn/v1"
```

### 问题 3：图片分析结果不准确

**可能原因**：
- 图片质量太低
- 图片内容不清晰
- 商品类型不常见

**解决方法**：
- 上传高清图片（推荐 800x800 以上）
- 确保商品在图片中清晰可见
- 添加文字描述帮助识别

### 问题 4：上传大图片很慢

**可能原因**：
- 图片文件太大（> 5MB）
- 网络速度慢

**解决方法**：
- 压缩图片到 < 2MB
- 使用网络 URL 代替上传
