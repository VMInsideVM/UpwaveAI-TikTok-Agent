# Excel 输出目录

此目录用于存放 TikTok 达人推荐 Agent 生成的 Excel 文件。

## 文件命名格式

```
tiktok_达人推荐_{商品名}_{日期时间}.xlsx
```

示例:
- `tiktok_达人推荐_口红_20250103_143025.xlsx`
- `tiktok_达人推荐_运动鞋_20250103_150512.xlsx`

## Excel 内容

每个 Excel 文件包含一个工作表,包含以下信息:
- 达人基本信息(账号名、粉丝数等)
- 近期数据(涨粉数、播放量等)
- 带货数据(销量、互动率等)
- 联系方式(如有)

## 注意事项

- Excel 文件会自动去重(保留第一次出现的达人)
- 如果使用多维度排序,数据会自动合并
- 建议定期清理旧文件以节省空间
