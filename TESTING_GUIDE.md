## 🔍 API 测试（使用 curl）

### 健康检查
```bash
curl http://127.0.0.1:8001/api/health
```

### 登录获取 Token
```bash
curl -X POST http://127.0.0.1:8001/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"***REMOVED***"}'
```

### 生成邀请码（需要 Admin Token）
```bash
TOKEN="your_access_token"
curl -X POST http://127.0.0.1:8001/api/admin/invitation-codes \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"count":3}'
```

### 查看所有用户
```bash
TOKEN="your_access_token"
curl http://127.0.0.1:8001/api/admin/users \
  -H "Authorization: Bearer $TOKEN"
```

### 创建会话
```bash
TOKEN="your_access_token"
curl -X POST http://127.0.0.1:8001/api/sessions \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json"
```

## 🐛 已知问题和限制

### 1. Bcrypt 版本警告
**问题**: 启动时显示 `error reading bcrypt version`

**影响**: 无（仅警告，不影响功能）

**解决**: 可忽略，或升级 bcrypt 包

### 2. WebSocket 连接需要 Token
**问题**: WebSocket 必须通过 query 参数传递 token

**解决**: 前端已正确实现 `ws://host/ws/{session_id}?token={jwt_token}`

### 3. 图片上传功能
**状态**: 原代码中有图片上传功能，但新界面中暂未实现

**建议**: 如需要，可在后续版本添加

### 4. 历史会话加载
**状态**: 聊天历史列表可显示，但点击加载功能未实现

**建议**: 需要后端添加加载历史消息的接口
