# 充值按钮修复说明

## 问题描述

**原始问题**：
用户在积分不足时点击"去充值"或"立即充值"按钮后，充值界面的浮窗没有弹出。

## 根本原因

**错误的函数调用**：
- 按钮调用了 `showRechargeOptions()`
- 但实际存在的函数是 `openRechargeModal()`

### 问题代码位置

#### 位置 1：确认弹窗中的"去充值"按钮
**文件**：[static/index.html:1791](static/index.html#L1791)

**修复前** ❌：
```html
<button id="rechargeFromModalBtn" onclick="showRechargeOptions(); closeConfirmGenerateModal();">
    💳 去充值
</button>
```

**修复后** ✅：
```html
<button id="rechargeFromModalBtn" onclick="openRechargeModal(); closeConfirmGenerateModal();">
    💳 去充值
</button>
```

#### 位置 2：聊天界面中的"立即充值"按钮
**文件**：[static/index.html:3283](static/index.html#L3283)

**修复前** ❌：
```html
<button onclick="showRechargeOptions()">
    💳 立即充值
</button>
```

**修复后** ✅：
```html
<button onclick="openRechargeModal()">
    💳 立即充值
</button>
```

## 正确的函数

**函数名称**：`openRechargeModal()`

**函数位置**：[static/index.html:3732](static/index.html#L3732)

**函数代码**：
```javascript
async function openRechargeModal() {
    const modal = document.getElementById('rechargeModal');
    modal.style.display = 'flex';
    modal.classList.add('show');
    // 重置状态
    // ...
}
```
